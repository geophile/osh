 18  """C{installosh INSTALL_DIRECTORY [OSH_CONFIG_FILE]} 
 19   
 20  Installs osh on a cluster. The cluster is identified using 
 21  remote execution syntax, for example:: 
 22   
 23      osh @fred [ installosh /usr/local/lib/python2.6/site-packages ] 
 24   
 25  C{INSTALL_DIRECTORY} specifies the installation directory on each node 
 26  of the cluster. On each node of the cluster, a C{.oshrc} files is 
 27  created in the home directory of the user specified for 
 28  C{CLUSTER}. This file is copied from C{OSH_CONFIG_FILE}, if 
 29  specified. Otherwise, C{~/.oshrc} is copied. 
 30   
 31  Not available through the API.
